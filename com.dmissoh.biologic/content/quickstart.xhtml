Welcome to BioLogic

Quick start

This tutorial describes the main functionalities of this software.

After steps I) and II) you are able to configure and run you first sequence. Steps III)-V) shows some advanced functionalities.


The picture above shows the toolbar of BioLogic. Following buttons are available:
  • (1) The 'New Sequence View...'
  • (2) The 'Save' button
  • (3) The 'Set Key Bindings...' button
  • (4) The 'Export as CSV' button

I) Configure at least one event

Define the events and define the hot keys to use to record them

First of all, you want to tell BioLogic which kind of event you want to record.
You can either directly click on the [Set Key Bindings...] button in the toolbar or access the same configuration page through using the menu Window>Preferences.


For you first run, just click on the [New...] button and than click on [OK] to store you first Event Key Binding setting.

In this simple example, you create a new event that can be started using the keyboard key [A] and stopped using the keyboard key [S].
This is obviously not a punctual event, i.e. an event without duration.

Choose the time unit

On the same preference page, you can choose the time unit that the system should use to measure time differences (e.g. for the CSV export).

II) Record a sequence

Starting a sequence

To start a new sequence with the event configurations you done in step I), click on the [New Sequence View...] button .
The following dialog pop up:


Live and Recorded Mode

Sequences can be started in a two different mode: Live and Recorded.
In the Live mode, the current time is logged. In the recorded mode, a running sequence can be paused, and only the elapsed time is recorded.

Enter a short description or click the [OK] button to use the default description.

Control the running sequence

s A new view is opened and the new sequence can now been started by clicking the start/stop button .
Now use the keys you define in step I) to start and end events (in this case the keys [A] and [S]).

During the recording, the events are logged in the Running events view (located in the left panel of the application).


This view list all running event (i.e. the ones that havn't been closed) with their corresponding start and end hot keys.

At the end of your recording sequence, terminate your sequence by clicking on the start/stop button again.

Every sequence is stored, and can be opened again by selecting it from the Logs view. This view is also located in the left panel of the application.


Actions in the toolbar of the Log View give you the possibility to delete all log files, open the folder where they are stored or refresh the view.

Running concurrent sequences

BioLogic gives you the possibility to record more than one sequences at the same time. You can toggle between those instances with the following shortcut [Shift] + [Tab].

III) Exporting a sequence

The sequence you previously recorded can be exported as a CSV-File (Comma Separated Values) which can easily been opened in Excel.
Two type of exports are available and are accessible by clicking on the small down-arrow that is located close to the export button :

  • Export log entries
  • Export grouped events
These commands are self explained. Just try them.

IV) Backup functionalities

BioLogic offers an simple backup mechanism to backup all your work and export the result as a compressed file (also known as zip files). By calling this command that is available over menu Utilities>Backup all sequences as well as the csv files you generate from them are exported.
